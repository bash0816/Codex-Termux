# Codex-Termux

Codex wrapper package for Termux.

Termux 向け Codex wrapper package です。

## ⚠️ v0.130.0 / v0.131.0 をお使いの方へ

v0.130.0 に含まれるバイナリに不具合があります。また v0.131.0 は `codex update` が動作しない不具合があります。v0.131.1 で修正済みです。以下のコマンドで更新してください。

```sh
npm install -g @bash0816/codex-termux@latest --force
```
